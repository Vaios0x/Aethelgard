## Contribuir

Gracias por tu interés en Aethelgard.

1. Haz fork y crea una rama: `feat/mi-cambio`
2. Ejecuta: `npm i` en `Aeth/`, `contracts/` y `aethelgard-backend/`
3. Corre CI local: `Aeth: npm run build`, `contracts: npm run compile`, `backend: node index.js`
4. Sigue el [Código de Conducta](./CODE_OF_CONDUCT.md)
5. Abre un PR con descripción clara y screenshots/logs

### Issues
- Usa las plantillas (bug/feature)
- Incluye pasos para reproducir, entorno y versión
